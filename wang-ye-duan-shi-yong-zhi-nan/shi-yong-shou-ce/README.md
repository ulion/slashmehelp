# 3. 角色与圈子的日常管理

在新的自组织协作模式里，角色和子圈是非常重要的组成部分。一个团队的工作内容以定义不同角色/子圈的方式来呈现，而且这个定义是可以随着团队的需要而动态调整的。 每个伙伴可以一人同时承担多个角色，参与到不同类型的工作中来激发自主性。
